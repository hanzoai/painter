# Hanzo Studio Documentation

**The most powerful and modular visual AI engine and application.**

## Overview

Hanzo Studio is a node-based interface for designing and executing advanced AI workflows using graph/flowchart based interaction. It supports stable diffusion, video generation, audio synthesis, and more.

## Quick Start

### Installation

```bash
pip install hanzo-studio
```

### Basic Usage

```bash
python main.py
```

Access the web interface at `http://localhost:8188`

## Features

- **Visual Node Interface**: Design complex AI workflows without coding
- **Multiple AI Models**: Support for SD, SDXL, SD3, Flux, video models, and more
- **Extensible**: Custom nodes and plugins
- **Performance**: Smart memory management and execution optimization
- **Cross-Platform**: Works on Windows, Linux, and macOS
